About Me

A Computer Engineer who delivers secure, efficient, and purposeful solutions — not just code.

Computer Engineering graduate from UNIWA, Athens. I build full-stack web applications with security as a core discipline — every system I ship is also hardened against the attacks that expose data or take services offline. From protected databases and encrypted pipelines to Bash-automated infrastructure and IoT prototypes, I work across the full depth of a system — not just the layer users see.

New build, security audit, performance fix, or end-to-end development — I deliver complete, documented work that holds up under real-world conditions.

What I Bring

Web Development

End-to-end web applications using PHP, JavaScript, Java, Python, C++, HTML, and CSS — from a minimal prototype to a production system.

Databases

Secure, normalized database design with MySQL and MariaDB — properly indexed, performance-tuned, and guarded against injection.

Cybersecurity

OWASP-based threat modeling, SQLi and XSS prevention, buffer overflow mitigation, cryptography, and data encryption — applied, not theoretical.

Linux & Automation

Advanced Bash scripting for deployment pipelines, system monitoring, forensic analysis, and administrative automation workflows.

Metadata Systems

Descriptive, structural, and administrative metadata using TEI, METS, and custom schemas — for archives, research systems, and digital libraries.

IoT & Hardware

Hardware-to-cloud prototypes with Raspberry Pi, Sense HAT, and Alexa Voice Service — including my BSc thesis project.

Quality Assurance

Manual and automated testing with a TDD discipline — catching defects at the design stage, not after deployment.
